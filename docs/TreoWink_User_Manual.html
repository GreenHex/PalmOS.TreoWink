What is TreoWink?

TreoWink allows you to keep your Treo ON for a specified time, even when there is no user input. This is useful while playing games or viewing maps, for example.

Downloads

The latest version of TreoWink can be downloaded here. The latest version of this user manual can be viewed online here.

Installing TreoWink

HotSync or copy TreoWink.PRC to your Treo phone. It will apprear in the Preferences Panel under "Other". It should be installed in the internal memory for proper functioning.

Uninstalling TreoWink

Delete TreoWink from the [Home]->[Menu]->[Delete] window.

Compatibility

TreoWink should work as described with all current Treo series smartphones.

TreoWink has been developed and tested on the Cingular Treo 650 (Unlocked) with the latest updater.

Operation

Invoking TreoWink from the Preferences Panel will display the TreoWink Preferences form. The options are straightforward and self-explanatory.

Features of TreoWink

  • Tiny PRC (~ 9 KB), NO hooking to 'hede', though it uses 'hvch' notifications to trap key presses.
